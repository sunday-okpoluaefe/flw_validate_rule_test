## Installation
#### Step 1: Clone the repository

```bash
git clone https://github.com/sunday-okpoluaefe/flw_validate_rule_test.git
cd flw_validate_rule_test
```

#### Step 2: Install NPM packages
```bash
npm i
```

#### Step 3: Start in development mode
```bash
npm run start
```
